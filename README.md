# MovieArchiveIndex
本仓库用于系统化整理互联网上可公开访问的影视类网站与影视信息页面，重点围绕电影、电视剧、综艺、动漫等内容进行结构化记录。仓库内容主要包含站点基础信息、访问地址、内容类型说明以及相关页面用途描述，方便进行长期索引、对比与研究。本项目不存储、不提供任何影视资源文件，也不涉及在线播放或下载行为，仅作为公开信息的整理与归档使用。所有资料均来源于公开网络环境，适合用于影视站点研究、内容导航整理以及搜索引擎抓取行为分析等用途，强调信息完整性与可追溯性。
